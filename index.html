Rascunho do Projeto

é um sistema que compara o percentual de crescimento do salário mínimo e da inflação entre os anos de 2010 a 2020

além disso, o sistema permite listar o histórico desses dois valores separadamente.

O sistema vai ter 3 opções.

1 - Listar o historico de salario minímo de 2010 a 2020

2 - listar o indíce de inflação

3 - mostrar a comparação

O sistema vai perguntar qual a opção desejada (escolha)

de acordo com o selecionado o sistema retorna a informação.

Os dados são fornecidos no projeto
